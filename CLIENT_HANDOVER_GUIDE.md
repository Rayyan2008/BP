# Being Personalised - Client Handover & Administration Guide

Welcome! This guide will help you manage your Being Personalised website and handle customer enquiries.

## Quick Start: Admin Portal Access

### Step 1: Access the Admin Dashboard
1. Go to: `https://yourdomain.com/admin/login` (or `/admin/login` on localhost)
2. Enter your admin password when prompted
3. Click "Login"

### Step 2: You Should See
- "Being Personalised Admin" heading
- Three tabs: **Enquiries**, **Orders**, **Alert Banner**
- Enquiries counter showing number of new enquiries

---

## Managing Enquiries

### Viewing Enquiries
1. Click the **Enquiries** tab (it should be active by default)
2. **Left panel** shows all enquiries with:
   - Customer name
   - Email address
   - Occasion type
   - Status badge (color-coded)
3. **Right panel** is empty until you click an enquiry

### Selecting an Enquiry
1. Click any enquiry card in the left panel
2. The right panel fills with full details:
   - **Name & Contact Info**: Customer's full details
   - **Enquiry Message**: What they requested
   - **Occasion & Date**: When they need it
   - **Status Buttons**: Change current status
   - **Internal Notes**: Add private notes (not shared)
   - **Quick Actions**: Email, WhatsApp, Delete buttons

### Understanding Enquiry Status

| Status | Meaning | Action |
|--------|---------|--------|
| **New** | Just received | Review and plan response |
| **Contacted** | You've reached out | Awaiting customer response |
| **In Progress** | Work is underway | Keep notes updated |
| **Completed** | Order fulfilled | Archive for records |

### Changing Status
1. Select an enquiry
2. Click the desired status button (New, Contacted, In Progress, Completed)
3. Status updates instantly
4. Button appearance changes to show selected status

## Communicating with Customers

### Via WhatsApp (Recommended)
1. Select an enquiry
2. Click the **WhatsApp** button
3. Your phone's WhatsApp opens with customer's number
4. Send your message

### Via Email
1. Select an enquiry
2. Click the **Reply via Email** button
3. Your email client opens with customer's email pre-filled
4. Type your message and send

### Notes on Communication
- Keep customer's preferred contact method in mind
- Their preference is noted in the enquiry details
- Respond within 24 hours for best conversion
- Use WhatsApp for quick questions, email for quotes

## Using Internal Notes

### When to Add Notes
- After contacting a customer
- When pricing discussed
- If customer requested modifications
- Follow-up actions needed
- Any special requirements

### How to Add Notes
1. Select an enquiry
2. Scroll to "Internal Notes" section
3. Type your note in the text area
4. Click "Add Note"
5. Note is saved and displayed in the notes box

### Example Notes
- "Sent quote for custom frame - awaiting approval"
- "Customer preferred blush pink color - available"
- "Delivery needed by June 20th - feasible with express shipping"
- "Follow up if no response by June 15th"

## Filtering Enquiries

### By Status
1. Click **"Filter by Status"** dropdown
2. Options:
   - **All Enquiries**: Show everything
   - **New**: Unreviewed enquiries
   - **Contacted**: You've reached out
   - **In Progress**: Active orders
   - **Completed**: Finished orders
3. Counter shows how many in each category

### Using Filters
- Start your day by reviewing **New** enquiries
- Check **Contacted** to see pending responses
- Monitor **In Progress** daily
- Archive **Completed** occasionally

## Enquiry Workflow Example

### Day 1: New Enquiry Arrives
1. Status: **New**
2. Review details
3. Add internal note: "Reviewed - meets requirements"
4. Contact customer via preferred method

### Day 2: Customer Responds
1. Update status to: **Contacted**
2. Add note: "Discussed specifications - send quote"
3. Send pricing quote

### Day 3-5: Awaiting Confirmation
1. Keep checking **In Progress** section
2. If no response by Day 5, follow up
3. Add note about follow-up attempt

### Day 6: Order Confirmed
1. Update status to: **In Progress**
2. Add note: "Order confirmed - production starts"
3. Note production timeline

### Day 10+: Order Complete
1. Send customer notification
2. Update status to: **Completed**
3. Add note: "Order shipped - customer satisfied"

## Best Practices

### Response Time
- Aim to respond within **2 hours** (during business hours)
- Set a calendar reminder for unaddressed enquiries
- Use internal notes to track when you last replied

### Customer Service Excellence
- Always acknowledge receipt of enquiry
- Provide realistic timelines
- Be transparent about pricing
- Ask clarifying questions if needed
- Maintain professional, warm tone

### Enquiry Organization
- Use status consistently
- Add notes for every interaction
- Update status after each customer contact
- Clean up completed enquiries monthly
- Keep confidential information secure

### Common Objections & Solutions

| Objection | Solution |
|-----------|----------|
| "It's too expensive" | Discuss smaller/simpler options |
| "Can you do it faster?" | Check if expedited production possible |
| "Can I customize further?" | Use internal notes to explore options |
| "I want a discount" | Document in notes, decide case-by-case |

## Troubleshooting

### Can't Log In
- Check admin password is correct: `SapotoInfosys`
- Clear browser cookies and try again
- Try a different browser
- Contact technical support

### Enquiry Not Showing
- Refresh the page (F5 or Cmd+R)
- Check filter - might be hidden by status filter
- Clear browser cache
- Log out and log back in

### WhatsApp Button Not Working
- Check your phone has WhatsApp installed
- Verify phone number in system is correct
- Try from a different device
- Check internet connection

### Email Button Not Working
- Check default email client is configured
- Verify email address is correct format
- Try copying the email manually
- Use Gmail web interface as backup

## Security Reminders

### Protect Your Admin Access
- Never share your admin password
- Log out when leaving your computer
- Use a strong, unique password if you change it
- Clear browser history periodically

### Handling Customer Data
- Keep email addresses confidential
- Don't share phone numbers externally
- Use notes for internal team only
- Compliant with GDPR (UK) requirements

### Backups
- Your enquiries are saved locally in your browser
- Clear browser data might delete enquiries
- Recommend monthly export (future feature)
- Contact support for data recovery if needed
